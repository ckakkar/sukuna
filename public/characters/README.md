# Character Images

Place character PNG images in this directory with the following naming convention:

- `sukuna.png` - Ryomen Sukuna
- `gojo.png` - Satoru Gojo
- `yuji.png` - Yuji Itadori
- `yuta.png` - Yuta Okkotsu
- `toji.png` - Toji Fushiguro
- `todo.png` - Aoi Todo
- `kinjihakari.png` - Kinji Hakari
- `choso.png` - Choso

Images should be:
- PNG format
- Square aspect ratio recommended (1:1)
- High resolution (at least 512x512px)
- Transparent background preferred

If images are not found, the app will display a fallback with the character's first Japanese character.
